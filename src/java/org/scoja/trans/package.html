Connect

Squid proxy default listen port is 3128.

Unauthenticated connect example. Request: 


CONNECT mail.google.com:443 HTTP/1.1
User-Agent: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.1.19) Gecko/20081202 Iceweasel/2.0.0.19 (Debian-2.0.0.19-0etch1)
Proxy-Connection: keep-alive
Host: mail.google.com

Response: 

HTTP/1.0 200 Connection established
